Latency monitor

Measuring HTTP response times from the command line

Active Personal project 2024

Overview

A lightweight Python script that wraps curl via the subprocess module to measure total HTTP response time for any URL. Designed as a focused learning exercise in Python's standard library — process spawning, output capture, time formatting, and basic statistics — while producing genuinely useful output.

The script runs a configurable number of timed requests against a target endpoint, prints each result with a timestamp as it arrives, then summarises the run with an average latency across all samples.

How it works

01

Spawn curl

Uses subprocess.run() to invoke curl with the -w '%{time_total}' flag, suppressing output and capturing only the timing float.

02

Collect samples

Loops over n runs with a configurable delay between each, appending each parsed latency to a list.

03

Report & summarise

Prints a timestamped line per run, then computes and displays the average across all collected samples.

04

Handle interrupts

Wraps the loop in a try/except KeyboardInterrupt so partial results are always reported cleanly on Ctrl+C.

latency_monitor.py 
import subprocess
import time

# Configuration
url = "https://phil.beer"
cmd = ['curl', '-s', '-o', '/dev/null', '-w', '%{time_total}', url]
runs = 3
delay = 5
latencies = []

print(f"Monitoring latency for {url} ({runs} runs, {delay}s delay)...\n")

try:
    for i in range(1, runs + 1):
        # Run curl and capture only the number
        result = subprocess.run(cmd, capture_output=True, text=True, check=True)

        # Convert string output to float and store it
        latency = float(result.stdout.strip())
        latencies.append(latency)

        # Print the formatted line as requested
        timestamp = time.strftime("%H:%M:%S")
        print(f"[{i:02d}] {timestamp} - Total time:\t{latency:.6f}")

        if i < runs:
            time.sleep(delay)

except KeyboardInterrupt:
    print("\nInterrupted by user.")

# Calculate and display the average
if latencies:
    avg_latency = sum(latencies) / len(latencies)
    print("\n" + "="*30)
    print(f"Final Results for {url}")
    print(f"Total Runs:    {len(latencies)}")
    print(f"Average Time:  {avg_latency:.6f} seconds")
    print("="*30)
